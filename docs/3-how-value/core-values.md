# 坦诚、主动、渴望

> 核心观点：坦诚建立信任，主动带来控制感，渴望提供长期成长的动力。

## 建议结构

1. 先说明为什么只选这三个价值观。
2. 每个价值观只给一个定义、一个正例和一个反例。
3. 解释三者如何相互约束：只有渴望可能冒进，只有坦诚可能消极。
4. 把价值观翻译成招聘、协作和困难时刻的可观察行为。
5. 说明价值观冲突时你的排序原则。
