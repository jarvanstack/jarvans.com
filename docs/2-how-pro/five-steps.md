# 五步工作法

> 核心观点：正确顺序是质疑、删除、优化、加速、自动化；顺序错了，效率越高可能浪费越大。

## 五步

1. 质疑：这个要求为什么存在？
2. 删除：哪些范围不做也能达到目标？
3. 优化：剩余部分有哪些更好的方案？
4. 加速：如何缩短验证和交付周期？
5. 自动化：稳定、重复的步骤才值得自动化。

## 建议写法

用同一个真实需求贯穿五步。每一步写清“原方案、采取的动作、删除的成本、得到的结果”，最后对比直接开工会多花多少时间。
