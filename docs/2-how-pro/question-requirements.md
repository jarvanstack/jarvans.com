# 第一步：质疑每一项需求


拿到需求第一步，不要开始写代码，而是先质疑需求。

默认所有需求都不需要做。

你到了一个医院，跟医生说：“我有点感冒，给我开点抗生素。”医生通常不会马上把抗生素给你，他会先确认你是病毒性感冒还是细菌性。因为对于病毒性抗生素是没用的。
