# 工程师的“五个等级”，你处在哪一级？ 

![五级工程师](https://cdn.jarvans.com/blog/202320260712175526682.png)


![对应大厂职级](https://cdn.jarvans.com/blog/202320260712202503287.png)


**五级工程师的定义是每一级的价值比上一级高 10 倍, 是数量级上的差距**

## 六级：尚未形成独立工程闭环(1-2年)

无法独立负责生产项目, 毕业生, 或者初级程序员, 对应阿里 P4

> 校招一般需要到六级工程师


## 五级：能够独立解决工程问题(3-5年)

五级工程师: 独立负责生产项目, 高级工程师(3-5年), 对应阿里 P5, P6, P7 


掌握分析解决问题的能力，主动协调资源，独⽴处理复杂问题，拿到有效结果。

> 社招一般需要到五级工程师, 是普通人的天花板


### 什么不是五级工程师?

* 码农: 工龄虽然长, 但是每天重复性的工作, 没有独立解决复杂问题

## 四级：能带领其他工程师完成重要工作

对于第四级的工程师，就需要有**领导能力**和在工程上把大问题化解为小问题的能力。

<br/>

四级工程师对业务目标理解深刻, 在现有的条件下平衡各种方案的投入产出, trade-off

<br/>

比如在建造一座海湾大桥时，工程师会在现有资金的条件下，根据交通的需求设计一个200年使用寿命的大桥，但是为了让军队迅速通过一条河，他们追求的目标就变成了在最短时间内建造一座足够让军队安全渡河的浮桥。

<br/>

目标不同，工程师的解决方案就不同，这件事对于土木工程师和桥梁工程师来讲，常常不是问题，但是很多程序员，常常会把海湾大桥修成浮桥，也会把浮桥按照海湾大桥来慢慢修。

<br/>

> 四级工程师能有百万年薪, **是没有机遇, 仅靠努力和能力达到的天花板**

<br/>

**三级光靠努力和能力是不够的。更重要的，看机遇**
