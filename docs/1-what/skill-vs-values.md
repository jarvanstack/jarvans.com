# 专业能力 VS 价值观, 哪个更重要?

![](https://cdn.jarvans.com/blog/202320260712164221244.png)

## 1. 能力好

好的程序员不是干了 10 年的老程序员, 也不是搞技术的极客

而是具有职业素养 (professional), 像是一个脑外科医生一样

<br/>

**好的脑外科医生不是过去完成 1 个复杂手术, 而是做了大量手术, 成功率都很高**

且脑外科医生知道针对不同情况如何提高成功率

<br/>

所以对于程序员来说, 思维方式很重要

怎样分析问题，找了怎样的资料，采取了怎样的步骤，此后做了哪些措施来避免这种错误再次出现

## 2. 价值观好

我关心价值观 TOP 3 分别是: **坦诚, 主动, 渴望**

一个人的未来成就(有限样本): **能力占 40%, 价值观占 60%**

<br/>

因为你能轻易发现, 有许多能力强但是价值观不够, 最后成就一般

但你很难找到一个人, 能力很强但是价值观不够, 最后成就一般的

<br/>

**因为有这种价值观的人能力很难平庸**

> 每个价值观的上限都很高, 没法做到完美, 我们的目标就只做的比之前好一些, 共勉

### 2.1. 坦诚

坦诚反面是不坦诚, 包括对自己的不坦诚和对别人的不坦诚

<br/>

我见过的大部分企业家和公司都把坦诚作为最重要的价值观

包括: 段永平, 黄峥, 马化腾, 马云

**因为坦诚是沟通的基石**

### 2.2. 积极主动

积极主动意味着: **你的未来由你自己主动控制, 而不是别人**

<br/>

会主动解决问题, 主动承担更多, 往往会付出更多

可能有些人会说这是在 PUA

你是不是想要压榨打工人的劳动力?

老板给我那么少的钱, 我就只干我分内的活就行了, 为什么要让我做更多?

<br/>

因为我觉得这是提升能力的最快的路, 能力提升了, 价值也就提升了

价格会围绕价值波动

其实推荐的做法是: **把自己当老板看**

**因为把自己当老板看，像老板一样拼命干活，能力自然就提高了。有了能力，假如多玩不能给你好的回报，其他公司一定会给。**

<br/>

把自己当老板看, 很多事情就豁然开朗了, 因为你同时有老板视角和员工视角, 能看到很多东西

### 2.3. 渴望


#### 2.3.1. 眼睛里有光的人

一次跟 CEO 吃饭

他说: 如果一个项目第一笔钱花完了, 团队应该解散, 但是如果制作人来争取, 我们会考虑要不要再给一笔钱

我问: 怎么判断

他说: 主要看制作人眼睛里有没有光, 我们面试制作人的时候也主要看制作人眼睛里有没有光, 我们之前也看其他东西, 但发现这个最有效

#### 2.3.2. 持续强烈的愿望

活法里有这样一段话:

想做成一件事，首先要有强烈的愿望，甚至达到痴迷的程度。“愿望强烈的程度促使你睡也想，醒也想，一天24小时不断的思考，透彻的思考，从头顶到脚底，全身充满了这种愿望，如果从身上某处切开，流出来的不是血，而是这种愿望”

<br/>

张一鸣读完发了个微博, 觉得这太难了

<br/>

有些人可能觉得我在说什么玄学,

心想事成, 怎么可能?

#### 2.3.3. 大脑里面有 5 个 CPU

你是不是经常能看到有人走路拿手机看, 电梯拿手机看, 上厕所手上也得有点什么?

为什么会这样呢?

<br/>

想象一下大脑里面有 5 个 CPU, 大脑会尽量把 CPU 用满

- 大脑首先是不关机: 它在你睡觉的时候也运行, 因为你会做梦
- 大脑可以待机: 睡觉和冥想的时候相当于待机, 运行简单任务
- 其他时候, 大脑会尽量让你的 CPU 满载, 如果发现你的 CPU 没有满载, 他就会后台运行其他程序

所以你走路时候, CPU 只利用了 20%, 大脑总是希望 CPU 满载运行, 于是你会刷手机, 想事情

<br/>

当你专注一件事, 且有持续强烈的愿望, 你就能把空闲的算力用上

可能几天后, 你来了解决问题的灵感, **其实不是你突然想到, 大脑已经在后台帮你算了很久了**
