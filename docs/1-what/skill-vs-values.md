# 能力 VS 品质, 哪个更重要?

![](https://cdn.jarvans.com/blog/202320260712164221244.png)



我观察的结论: 成功的人, 能力占 40%, 品质占 60%



## 1. 思考一下, 什么是能力好?


不是一个干了 10 年的老程序员就能力好，也不是一个搞技术的极客能力好。


<br/>
而是具有职业的素养，professional，像是一个脑外科医生一样。

**好的脑外科医生不是过去完成了一场复杂的手术，而是做了大量的手术，而且成功率都很高。**

因为他知道针对不同的情况，如何提高手术的成功率。

<br/>

程序员在入行的时候，他的 title 就叫做工程师。

一个土木工程师，他会知道现在的形势是什么，我们有多少资源，决定是造一个快速过河的浮桥，还是坚固一些的石桥，还是一个百年的海湾大桥。

但我见过很多程序员，他们会把海湾大桥建成浮桥，然后在需要浮桥的时候，又把它当做海湾大桥来慢慢建。

## 2. 什么是品质好？


<br/>

我最关心的三个品质分别是**坦诚、主动和渴望。**

<br/>

因为有这三种品质的人，是很难平庸的。

### 2.1. 第一个品质: 坦诚



第一个品质是坦诚。我见过的大部分做得好的企业，都会把坦诚作为他们最重要的品质，是他们的红线。你可以想象一下，如果我跟你说话，你知道我的话里面有 10% 是假话，你会怎么办？

### 2.2. 第二个品质是主动。

主动意味着**你的未来是由你自己控制的，而不是别人。**


<br/>


所以当遇到一个问题的时候，你需要尽量自己去控制这个问题的结果，而不是交给别人控制。所以你需要主动地解决问题啊，往往会更辛苦。

可能有人会说，我这是在 PUA，是不是想压榨工人的劳动力？“老板给我这么少的钱，我就只干我分内的活就行了，我为什么要做这么多？”



<br/>

因为我觉得主动地去掌控更多东西，负责更多东西，是提升能力最快的路。

**我一般推荐的做法是把自己当做老板看，在老板的角度去思考问题**

你同时拥有老板视角和员工视角，你的能力会提升非常快。

<br/>

**有了能力，假如公司不能给你很好的回报，其他公司也会给的。**




### 2.3. 渴望

第三个品质是渴望，我这里讲一个故事。

有一次跟 CEO 一起出去吃饭，他说：“如果一个项目的钱花完了，团队就应该解散；但是如果负责人过来争取，我们也会考虑要不要再给他第二笔钱。”

我问怎么判断。

他说，主要看负责人眼睛里面有没有光。我们面试的时候也看这个，之前也看其他的东西，但是最后发现这个最有效。


<br/>


活法里有这样一段话:

想做成一件事，首先要有持续强烈的愿望, 这愿望促使你睡也想，醒也想，一天24小时不间断的思考，透彻的思考.

<br/>

张一鸣读完发了个微博, 觉得这太难了


**我觉得这是一条很难的路，但是它是一条最近的路。**

#### 2.3.3. 大脑里面有 5 个 CPU


是不是经常看到有人走路玩手机，电梯里玩手机，地铁玩手机，上厕所手上也得有点什么？有没有想过为什么会这样呢？

<br/>


想象一下，你的大脑里面有 5 个 CPU，大脑尽量的会让这 5 个 CPU 跑满。

有这样几个信息：
* 第一，大脑是不关机的。你睡觉的时候它也在运行，因为你会做梦。
* 第二，大脑可以待机。你睡觉或者冥想的时候，它以低功耗运行，这就是待机。
* 第三，其他时候他就会尽量跑满。如果他发现你没有跑满，他就会想办法跑满。


比如你走路的时候，你的 CPU 只用了 20%。大脑发现你的 CPU 还没用满，就会想其他办法：你就会玩手机，跟人说话，或者想其他的东西。


<br/>

当你渴望做成一件事情，有持续强烈的愿望，你就能把这空闲的算力也用上。

3 天后，你突然对某个问题有了灵感。**其实不是突然的灵感，而是你的大脑已经在背后给你算了 3 天了。**
