# 专业能力还是价值观？

> 核心观点：招聘时，能力决定一个人现在能做什么，价值观决定他未来会如何选择和成长。

![](https://cdn.jarvans.com/blog/202320260712141434920.png)

## 当前草稿

### 能力好

不仅是写过很多代码、经验丰富，而是具有职业素养（professional），像脑外科医生一样对专业判断和结果负责。

### 价值观好

我最关心的三个价值观是：**坦诚、主动、渴望**。

我的观察是，有这些价值观的人，能力很难长期平庸。能力强但价值观不匹配的人，也可能因为一次次错误选择而限制自己的长期成就。

这只是当前判断，正文需要用真实案例、反例和边界继续验证，不能只给出“能力占 40%，价值观占 60%”这样的主观比例。

## 开场案例

写一个两难选择：候选人 A 技术强但不坦诚，候选人 B 能力稍弱但主动、诚实且学习快。说明你最后如何选择，以及后来发生了什么。

## 建议论证

1. 先定义“专业能力”，不要把它等同于写过很多代码。
2. 再定义“价值观”，聚焦坦诚、主动、渴望三个可观察行为。
3. 分别讨论短期交付和长期合作，不要用一个比例替代论证。
4. 给出不能妥协的红线，以及可以通过培养补足的能力。

## 反方观点与边界

不要绕过这些质疑，正文需要正面回答：

- 强调“主动承担”，会不会变成公司压榨员工的 PUA？
- 回报明显不足时，员工为什么还要承担更多？
- 主动、服从和没有边界感有什么区别？
- 如果组织不奖励正确的价值观，个人应该改变组织还是选择离开？

可以把边界落在一句话上：主动意味着为自己的选择负责，不意味着无条件替组织承担责任。
